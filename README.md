# ScapingForLunch
## 🎮 게임 소개
제작자의 실제 경험에서 비롯한 갓겜
밥도 제시간에 안 먹고 코인창부터 들여다 보는 당신, 과연 오늘 점심은 뭐가 될까요?

## ⚠️ 게임 규칙

시드머니: 1,000,000원으로 시작합니다.
제한 시간: 단 3분입니다. 장이 마감되면 강제로 정산됩니다.
상장 폐지: 주가가 300원 미만으로 떨어지면 그 즉시 상장폐지되며 보유 자산은 증발합니다.
파산: 자산이 5,000원 미만이 되면 게임 오버입니다.

## 🛠️ 기술 스택
- HTML5 / CSS3 (Tailwind CSS)
- JavaScript (Vanilla ES6+)
